Это моя программа для парсинга XML файлов определённого формата и записи результата в БД PostgreSQL.
Перед первым запуском необходимо использовать backup файл "PlantsDB" в любой пустой БД PostgreSQL. Название БД значения не имеет.
Создал свой backup файл, т.к. тот что в задании не задаёт первичный ключ для таблицы f_cat_plants. Также было необходимо сделать уникальными поле uuid таблицы d_cat_catalog и поле common таблицы f_cat_plants.
Проект собирается с помощью Maven командой "mvn clean compile assembly:single", создающей jar файл.
После запуска программы в диалоговом окне необходимо выбрать xml-файлы для парсинга. Допускается выбор нескольких файлов одновременно.
Все атрибуты каталога являются обязательными. Каталог игнорируется вместе с вложенными расстениями, если не хватает каких-либо атрибутов или если такой каталог уже есть в БД (определяется по полю uuid).
Все данные о расстении являются обязательными. Расстение игнорируется, если не хватает каких-либо данных или если такое расстение уже есть в другом каталоге (определяется по полю common).
